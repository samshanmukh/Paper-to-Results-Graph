About
Papers are extracted into claims and methods. Methods are turned into runnable code, executed in Daytona sandboxes, and each result attaches back to the graph as a Run with a VALIDATES / REFUTES verdict computed from measured metrics. Dashed amber arcs are papers contradicting each other. This page renders the live research graph mirrored to Butterbase.
